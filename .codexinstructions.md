# SPC Web Application

Tech Stack:
- Flask
- Pandas
- NumPy
- Plotly
- HTML
- Bootstrap

Current Features:
- Xbar-R Chart
- Xbar-S Chart
- 4 Attribute charts (P, Np, C, U)
- Cpk/Cp Analysis

Development Rules:
- Never break existing chart logic.
- Always provide complete file updates.
- Maintain backward compatibility.
- Use Bootstrap for UI components.
- Keep mathematical calculations separate from routes.
- Use pandas for data processing.
- Use Plotly for visualization.

Future Roadmap:
- User Authentication
- SAP Integration
- Scheduled Data Pulling
- AI Insights
- Western Electric Rules
- Multi-Plant Dashboard
- PDF Reports
